# ModbusRtuOverTcpSimulator

---
ModbusRtuOverTcpSimulator 是一个轻量级项目，通过 TCP 模拟 Modbus RTU 行为。  
**重要提示：** 本项目 **并非通常意义上的 RTU-over-TCP（Modbus TCP）**。  
仅用于 **教育目的和 RTU 初步学习参考**，展示 RTU 帧结构、逐字节发送时序及基本读写操作。

**注意：** 本项目适合学习、调试和演示 RTU 原理，**非生产级使用**。

---
**English:**

ModbusRtuOverTcpSimulator is a lightweight project that simulates Modbus RTU behavior over TCP.  
**Important:** This is **not the standard RTU-over-TCP implementation** (Modbus TCP).  
It is intended for **educational purposes and initial RTU learning reference**, demonstrating RTU frame structure, byte-level timing, and basic read/write operations.

**Note:** This project is suitable for learning, debugging, and showcasing RTU principles, but it is **not production-grade**.
